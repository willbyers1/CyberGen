## ğŸ“– Overview

**CyberGen** is a robust, serverless password generation utility designed for users who prioritize privacy and security. Unlike many online tools that process data on servers, **CyberGen runs entirely in your browser.**

Built with **Vite** and **TypeScript**, it ensures strict type safety and high performance. It solves the problem of creating strong, complex passwords while providing immediate visual feedback on password strength using a modern algorithm.

The project features a **Cyberpunk/Dark Mode** aesthetic with a responsive Glassmorphism UI, fully optimized for a seamless user experience.
